handle_embeddedRDF(node, graph, state)

source code 

Handles embedded RDF. There are two possibilities:

  • the file is one of the XML dialects that allows for an embedded RDF/XML portion. See the host.accept_embedded_rdf_xml for those (a typical example is SVG).
  • the file is HTML and there is a turtle portion in the <script> element with type text/turtle.
Parameters:
  • node - a DOM node for the top level element
  • graph (RDFLib's Graph object instance) - target rdf graph
  • state (state.ExecutionContext) - the inherited state (namespaces, lang, etc)
Returns: Boolean
whether an RDF/XML or turtle content has been detected or not. If TRUE, the RDFa processing should not occur on the node and its descendents.
