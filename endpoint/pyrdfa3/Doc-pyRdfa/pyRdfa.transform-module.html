Package transform

source code

Transformer sub-package for the pyRdfa package. It contains modules with transformer functions; each may be invoked by pyRdfa to transform the dom tree before the "real" RDfa processing.

Requires: RDFLib package

Organization: World Wide Web Consortium

Author: Ivan Herman

License: This software is available for use under the W3C® SOFTWARE NOTICE AND LICENSE

Version: 3.0

empty_safe_curie(node, options, state)

source code 

Remove the attributes whose value is an empty safe curie. It also adds an 'artificial' flag, ie, an attribute (called 'emptysc') into the node to signal that there _is_ an attribute with an ignored safe curie value. The name of the attribute is 'about_pruned' or 'resource_pruned'.

Parameters:
  • node - a DOM node for the top level element
  • options (Options) - invocation options
  • state (State) - top level execution state

vocab_for_role(node, options, state)

source code 

The value of the @role attribute (defined separately in the Role Attribute Specification Lite) should be as if a @vocab value to the XHTML vocabulary was defined for it. This method turns all terms in role attributes into full URI-s, so that this would not be an issue for the run-time.

Parameters:
  • node - a DOM node for the top level element
  • options (Options) - invocation options
  • state (State) - top level execution state
