Module termorcurie

source code

Management of vocabularies, terms, and their mapping to URI-s. The main class of this module (TermOrCurie) is, conceptually, part of the overall state of processing at a node (state.ExecutionContext) but putting it into a separate module makes it easider to maintain.

Requires: RDFLib package

Organization: World Wide Web Consortium

Author: Ivan Herman
