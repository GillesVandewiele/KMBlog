Implementation of the @property value handling.

RDFa 1.0 and RDFa 1.1 are fairly different. RDFa 1.0 generates only literals, see RDFa Task Force's wiki page for the details. On the other hand, RDFa 1.1, beyond literals, can also generate URI references. Hence the duplicate method in the ProcessProperty class, one for RDFa 1.0 and the other for RDFa 1.1.
